Project Overview

I was tasked with creating a new login page for our conference room guest wifi. This was intended to replace an existing wifi default page provided by a service.

Project duration November 2018 - November 2018

My role in the project was that of a UX UI Designer and Developer.

User Research

I researched reviews of existing wifi login pages online. This helped me to understand the most common pain points users experienced when logging into a wifi portal.

Pain Points

Confusing

The design needs to be simple and easy to use.

Mobile

Responsive and Mobile friendly

Light

This site needs to be a simple html page and shouldn't include a lot of javascript packages

Address Issues

Design

Executives selected the above design of the three options presented

Accessibility

All content was WCAG AAA accessible and tested

New Page

The new layout was very pleasing on the eyes and easy for users to understand

Launch

I compiled this site into a single html file as per the requirements provided to me. The page was relatively small and easy to implement within the provided wifi gateway software.

Take Aways

Impact

Updating our conference room wifi login page gave the department a professional presence. This was seen by all visitors as well as visiting vendors.

What I Learned

The really pushed me to my limits with CSS and understanding how to overlay images and make the site responsive and mobile friendly.
