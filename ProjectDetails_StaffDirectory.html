Project Overview

My current assignment doesn't have exact guidelines, there are times which I'm a Developer, Web Master, or Database administrator.

The purpose of this project was driven by internal needs. I was often tasked with an assignment which required I grant or remove access to an application for a user. However the user information wasn't always clear. I would spend time gradually acquiring bits of information from different databases until I had all the user information I needed.

Project duration November 2020 - November 2020

My role in the project was that of a UX UI Designer and Developer.

User Research

The current method of identifying a person required many steps and connections to databases to run queries for bits of information needed to conduct additional queries. It was time to build something better.

Pain Points

Slow

It was an inefficient use of my time to piece together bits of information from several sources.

Tedious

I'd have to connect to 2 -3 database to find bits of information that would lead to the actual information I need.

Unsure

I'd then have to verify the person with the department before proceeding.

Address Issues

Design

Something I can host on a Development server that will be easy to pull up and use.

Secure

I made sure to pull information which wouldn't create any security concerns.

Work Efficently

So much easier than hunting down all that information

Launch

I was able to publish this to our development server for myself and other developers to use when needed. This does not store any information, so it did not require any database back end which was great.

Side note, a few key staff where granted access to the site and loved it so much they requested we share it with all permanent staff.

Take Aways

Impact

I was able to utilize my time more efficiently after building and publishing this application. Senior executives loved this website and requested access be granted to all permanent staff to use as well.

What I Learned

This was a great first project to build in .Net Core and helped us get our Development servers configured for core.
