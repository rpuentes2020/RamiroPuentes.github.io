Project Overview

After experiencing unsual performance spikes, we needed a landing page for Faculty to visit prior to reaching our school site.

Project duration December 10 2023 - December 11 2023

My role within this project is that of a Lead UX designer, UX Researcher, UX Tester and Developer.

User Research

User research was conducted which focused on finding examples of "Reminder" and "Warning" pages.

This single page site would be used in the event of unusually high traffice. I proceed to look for inspiration online knowing the page would need to be clear and grab attention.

  • Grab attention
  • Remind Faculty of student priority
  • Still allow Faculty to continue to site if needed
