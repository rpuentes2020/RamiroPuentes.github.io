Project Overview

The content engineer responsible for maintaining the Refco site reached out to me with a request to create a fresh new layout. He wanted a single page layout that looked new and was responsive.

Project duration March 2019 - April 2019

My role in the project was that of a UX UI Designer and Developer.

User Research

The current website had not been updated since it's inception. The organization was eager and ready for an update to their existing website. The current layout was not responsive or mobile friendly. The primary visitors were elderly retired people, so ensuring the website was easy to navigate and read was top priority.

Pain Points

No Mobile

The current website did not offer any mobile friendly version

Accessibility

Accessibility was another priority and the existing site did not meet any WCAG Standards

Fresh from the 80's

Outdated look and feel

Address Issues

Design

A single page layout would make it easier for elderly to navigate the site

Accessibility

Ensure the new site will meet WCAG AA criteria

New Features

Ability to visit the new website on their mobile devices

Launch

The new layout was a smashing success. The stakeholders loved it and content engineer was happy with how easy it would be to maintain. The content engineer later reached out to me after the new site launch to let me know that the site was also a hit with visitors.

Take Aways

Impact

Updates made to the site made it responsive, mobile friendly and accessible. Adding an easy workflow for the content engineer to follow when making updates.

What I Learned

This helped me explore the available features a website has when visited through a mobile device. I enjoyed working with Refco and hope to continue working with them in the future
